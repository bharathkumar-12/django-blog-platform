# Blog-Application

Blog Application is a Python Blog App for Creating and Viewing posts.This is blog created by Django framework with simple blog functionality and some example content.

## Installation
```python
**Packages**
django = "== 3.0"
gunicorn = "== 19.9.0"
whitenoise = "== 4.1.4"

**Python**
python_version = "3.8"
```
